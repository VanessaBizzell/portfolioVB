Projects

Some of my latest work...

Creative Writing App

Creative Writing app that provides a daily writing prompt and creates a library of author's work. Created using Javascript in a React/NextJS framework with tailwindCSS. API built using Express and MongoDB.

Soundwave Music Sharing App

Music Sharing app built using Typescript in an Angular Framework. API built using Typescript/Express and a MongoDB database. Allows users to upload their own music and share with others for comments and likes.

Weather App

Weather app that displays weather by selected location. Built using javascript, HTML and tailwindCSS, the app utilises the OpenWeather API to provide current and forcast weather data in a user-friendly display.

Fantasy Book Map Quiz

Fun quiz to test your knowledge of book maps from the Fantasy genre. Created using Javascript, HTML and tailwindCSS.

Castle Text Adventure Game

Text-only adventure game - step back in time! Using object orientated programming principles (OOP). Created using Javascript, HTML and tailwindCSS.

Social Media App

A simple social media app that allows the user to post images with commentary and receive likes. This was built using Javascript and React in the NextJS framework.
