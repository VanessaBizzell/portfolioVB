Fantasy Book Map Quiz

❰❰❰ Portfolio Next Project ❱❱❱

Project Design

The brief for this was to create an interactive quiz using Javascript. I chose a visual quiz subject to engage the user to test their knowledge of book maps from popular fantasy books. It has proved popular with friends and their children.

Responsive Design

My book map quiz is designed to work on a variety of screen sizes. The questions stack below the map images on smaller screen sizes.

Accessibility and readability

My code is easily legible and accessibility has been acheived through good alt-text descriptions. Colour contrast has also been carefully considered.
