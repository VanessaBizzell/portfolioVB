Sapphire Castle [Adventure Text Game]

❰❰❰ Portfolio

Project Design

The brief for this project was to create a text-only adventure game using OOP programming principles in Javascript. I used whiteboarding software to create a clear game design strategy. My younger testers have overcome their initial scepticism at the lack of graphics to enjoy the retro gaming experience!

Responsive Design

Sapphire Castle has a very basic UI as part of the design concept. However, it does work well on different screen sizes.

Readability

My code is easy to read and has concise, clear comments to allow other developers to understand functions.
